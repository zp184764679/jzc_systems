### 2. 需要修改的API

#### Collapse组件（已废弃children方式）
**旧API (OrderList.jsx等)**:
```jsx
<Collapse>
  <Panel header="标题1">内容1</Panel>
  <Panel header="标题2">内容2</Panel>
</Collapse>
```

**新API**:
```jsx
<Collapse items={[
  { key: '1', label: '标题1', children: '内容1' },
  { key: '2', label: '标题2', children: '内容2' }
]} />
```

#### Input.Search addonAfter（已废弃）
**旧API**:
```jsx
<Input.Search addonAfter={<Button>按钮</Button>} />
```

**新API**:
```jsx
import { Space } from 'antd'
<Space.Compact>
  <Input.Search />
  <Button>按钮</Button>
</Space.Compact>
```

#### Spin tip属性
**问题**: tip只在fullscreen或嵌套模式下工作
**解决**:
```jsx
// 方式1: 使用fullscreen
<Spin tip="加载中..." fullscreen />

// 方式2: 嵌套使用
<Spin tip="加载中...">
  <div>内容</div>
</Spin>

// 方式3: 不使用tip
<Spin />
```
